DSpace

I planned and implemented a system of institutional repository for the university using DSpace framework, considering metadata to facilitate the searching inside the repository itself; allowing that the entire scientific and administrative production of the university could be accessed by the internet in an abroad way.

The implementation was part of the research paper I was doing.

In this project I had a lot of problems because the professors and researchers didn't agree about uploading their papers by free into a repository, so we had an impasse because of that. Then I had problems with the DSpace configuration.

DSpace is the framework made by MIT to be used in repository creation, it was not good looking, but customizable, and it was really hard to install and configure. In the end of 3 months of research, between papers read, and discussions about the plans, I got the framework working with the organization made.
